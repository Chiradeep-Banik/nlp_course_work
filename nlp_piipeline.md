# <p align="center">NLP pipeline</p>

## A NLP pipeline contains the following steps :
- Data Aquisition
- Text-preparation
    - Text cleanup
    - preprocessing
- Feature Engineering
    - TF-IDF
    - Bag of Words
    - Word2Vec
- Modelling
    - Model Building
    - Evaluation
- Deployment
    - Deploy
    - Monitor
    - Update
